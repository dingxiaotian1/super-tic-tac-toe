# 超级井字棋

一个现代化的在线井字棋游戏，支持人机对战和双人对战模式。

## 游戏特色

- 🔄 **两种游戏模式**：人机对战和双人对战
- 🤖 **智能AI对手**：优先进攻其次防守的AI算法
- ⏱️ **游戏时间统计**：实时显示游戏时长
- 📊 **已落子数量显示**：清晰展示当前棋盘状态
- ✨ **流畅动画效果**：
  - 第7枚落子时，第一枚棋子闪烁后消失
  - AI思考时的动态效果
  - 获胜时的黄色获胜线动画
- 📱 **响应式设计**：适配不同屏幕尺寸
- 🎨 **现代化UI**：圆形棋子设计，渐变色彩，卡片式布局

## 游戏规则

1. 3×3的标准九宫格
2. 率先连成3子者获胜
3. 棋盘最多同时显示6枚最新落子，第7枚落下时第1枚闪烁后消失
4. 横向、纵向或对角线形成3枚连续相同棋子即获胜

## 技术实现

- **前端技术**：HTML5 + CSS3 + JavaScript
- **UI框架**：Font Awesome图标库
- **动画效果**：CSS3动画 + JavaScript定时器
- **AI算法**：简单的决策树算法，优先考虑获胜，其次防守

## 使用方法

1. 克隆或下载项目到本地
2. 使用HTTP服务器运行项目，例如：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问 `http://localhost:8000`
4. 选择游戏模式开始游戏
